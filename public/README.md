# Public Assets

## Hero Background Image

**Required File**: `hero-bg.jpg`

### Specifications:

- **Dimensions**: Minimum 1920x1080px (Full HD)
- **Subject**: High-contrast fast food imagery (burgers, fries, etc.)
- **Style**: Dark, moody photography with rich colors
- **Format**: JPG or WebP
- **File Size**: Optimized for web (< 500KB recommended)

### Recommended Sources for Royalty-Free Images:

1. **Unsplash** (https://unsplash.com)
   - Search: "fast food dark", "burger photography", "street food"
2. **Pexels** (https://pexels.com)
   - Search: "fast food", "burger close up", "restaurant food"
3. **Pixabay** (https://pixabay.com)
   - Search: "fast food", "burger", "street food"

### Image Placement:

Place your hero background image at:

```
almaidah-cafe/public/hero-bg.jpg
```

### Fallback:

If no image is provided, the component will display the layered gradient background with the dark overlay effect.

### Optimization Tips:

- Use tools like TinyPNG or Squoosh to compress images
- Consider using WebP format for better compression
- Ensure the image has good contrast for text readability
- Dark or moody images work best with the multiply blend mode
